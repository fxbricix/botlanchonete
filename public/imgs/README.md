# Pasta de Imagens dos Mapas

Esta pasta contém as imagens dos mapas do CS2 no formato PNG.

## Mapas disponíveis:

- ancient.png
- de_anubis.png
- de_dust2.png
- de_inferno.png
- de_mirage.png
- de_nuke.png
- de_overpass.png
- de_train.png
- de_vertigo.png

## Formato das imagens:

- Formato: PNG
- Resolução: Variada (otimizada para web)
- Sistema de nomenclatura: Nome exato do mapa com prefixos quando necessário

## Como funciona:

1. A aplicação detecta automaticamente o mapa atual do servidor
2. Busca pela imagem correspondente na pasta
3. Exibe a imagem ou um placeholder se não encontrar
